## Unit Testing
We use py.test for unit testing. Install with `pip install -U pytest`.

To run tests, run `pytest tests`.

## Installation
### Dependencies
* Python3 
* SciPy
* NumPy
* Matplotlib
* Scikit-learn
* CVXPY
* CVXOPT (optional as a CVXPY solver)
* Jupyter Notebook (optional for notebook demo)
